# 🎬 YouTube 企业培训方案

## 操作步骤

### 1. 上传到 YouTube
- 登录 YouTube Studio
- 上传您的培训视频
- 设置可见性为"不公开"或"仅限链接"

### 2. 获取链接
- 复制视频链接，格式如：
  `https://www.youtube.com/watch?v=VIDEO_ID`

### 3. 在系统中配置
- 管理员界面 → 课程管理 → 编辑课程
- 粘贴 YouTube 链接到"飞书视频链接"字段
- 系统会自动识别并处理

## 优势
✅ 全球访问无障碍
✅ 无需VPN或特殊网络设置
✅ 高质量视频播放
✅ 可设置隐私保护

## 注意事项
- 选择"不公开"确保只有有链接的人能看到
- 企业账号可以更好的管理权限
