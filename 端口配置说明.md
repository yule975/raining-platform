## 🔧 配置文件位置

### 前端端口配置
- **文件**: `learn-bloom-pal-58-main/vite.config.ts`
- **配置项**: 
  ```typescript

    host: "0.0.0.0",
    port: 5173,  // 前端开发服务器端口
    open: true,
  },
  preview: {
    host: "0.0.0.0", 
    port: 5173,  // 前端预览服务器端口
  }
  ```

### 后端端口配置
- **文件**: `backend/src/server.ts`
- **配置项**: 
  ```typescript

  ```

## ⚠️ 注意事项

1. **端口占用**: 如果5173端口被其他应用占用，Vite会自动选择下一个可用端口
2. **防火墙**: 确保防火墙允许5173和3000端口
3. **浏览器缓存**: 更改端口后建议清理浏览器缓存
4. **书签更新**: 请更新浏览器书签到新的地址
