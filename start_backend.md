# 🚀 后端服务快速启动

## 如果需要启动后端服务（可选）

### 在终端中执行：

```bash
# 1. 进入后端目录
cd /Users/yule/Desktop/培训平台/backend

# 2. 安装依赖
npm install

# 3. 启动后端服务
npm run dev
```

### 预期结果：
```
Backend server running on http://localhost:3000
```
